Instance Addresses

An instance address creates a payment destination address for starting predefined OpenStack instances on this appliance. Instance addresses are automatically created on this appliance via callbacks initiated by the the {{ config['POOL_NAME'] }} site. Instances are not provisioned by this appliance until a payment has been verified as being sent by the end user to a corresponding instance address below.

A manual instance address can be created by clicking the Create Instance Address button. The resulting address can be used for testing purposes.

You can also get the JavaScript code for embedding an instance payment widget on a third party site by clicking on Embed Instance Payment Widget button. This can be used to sell services provided by this cluster independently of the pool operator.
