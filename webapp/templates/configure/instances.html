Instance Addresses

An instance address represents a payment destination used for starting a predefined OpenStack instance on this appliance. Bitcoin payments to an instance address are monitored by this appliance via Blockchain.info's API based payment callback service.

Instance addresses are automatically created on this appliance via callbacks initiated by the the {{ config['POOL_NAME'] }} site. Instances are not provisioned by this appliance until a payment has been verified as being sent by the end user to a corresponding instance input address below.

A manual instance address can be created by clicking the Create Instance Payment button. The resulting address can be used for testing purposes, promotion on a third party website or for exchanging compute resources with an untrusted entity.
