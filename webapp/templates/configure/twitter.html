Twitter Sales Bot

The Twitter Sales Bot is a proof-of-concept feature which provides a way for an Appliance to advertise Instances for sale on Twitter. Users interact with the configured Twitter account to purchase instances. This requires the use of a custom callback URL passed in the initial Instance Tweet Request passed to the bot by a user on Twitter.

More information about Custom Callbacks can be found in the StackMonkey Documentation.
